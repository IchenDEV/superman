# 重大决策记录

## 1. 博浪沙椎击秦始皇
- **决策**：散尽家财，结交刺客，在博浪沙刺杀秦始皇。
- **逻辑**：这是他早期作为“韩国贵族”的复仇本能，充满了游侠气息。虽失败，但展现了其极强的行动力和决绝。

## 2. 放弃景驹，追随刘邦
- **决策**：在寻找反秦势力的过程中，遇到刘邦，发现刘邦能听懂《太公兵法》，果断放弃投奔景驹，追随刘邦。
- **逻辑**：寻找能兼容自己“战略软件”的“硬件系统”。刘邦的纳谏能力是张良选择其作为“宿主”的核心原因。

## 3. 鸿门宴的危机公关
- **决策**：拉拢项伯，指导刘邦卑辞谢罪，成功化解项羽的杀机。
- **逻辑**：精准识别项羽阵营的裂痕（项伯），利用人情和联姻将其转化为己方资源，是顶级的统战与危机处理。

## 4. 下邑之谋
- **决策**：建议刘邦放弃关东大片土地，将利益分配给韩信、彭越、英布，构建反项羽统一战线。
- **逻辑**：舍弃短期领土利益，换取战略上的绝对优势，是“舍与得”的顶级博弈。

## 5. 功成身退，托辞辟谷
- **决策**：汉朝建立后，主动放弃三万户的封赏，只要了留县（初见刘邦之地），并托辞“辟谷”修仙，不问政事。
- **逻辑**：识别到帝国从“打天下”到“治天下”的系统底层逻辑变化，主动清零权力，实现完美避险。
