# 王阳明完整时间线

> 说明：标注 [存疑] 表示史料有争议或确切年份不确定；标注 [推测] 表示依据间接证据推断。

---

## 一、早年（1472–1499）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1472 | 出生于浙江余姚，原名王云，后改名王守仁，字伯安，号阳明。父王华，后官至礼部尚书。据传母亲孕期超常（14个月 [存疑]），祖母梦见天神送子，因此幼名"云"。 | 出身官宦书香之家，奠定儒学正统起点；异常诞生的传说日后被门人附会为"天纵之才"叙事。 |
| 1476 前后 | 5岁仍不能言，据传祖父王伦改名"守仁"（取《论语》"知及之，仁不能守之"之意）后方开口说话 [存疑]。 | "守仁"之名本身即一种哲学期许，暗示仁的内在持守，与后来心学主旨隐隐呼应。 |
| 1482–1483 | 父王华中状元（1481年）[存疑确切年份]，举家迁北京，阳明随行。早年广泛涉猎，习儒经、习骑射兵法，志向并不局限于科举。 | 接触多元知识体系，形成"成圣"而非仅"成官"的早期志向。 |
| 约 1489 | 拜访理学名儒娄谅（字克贞），娄谅告以"圣人可学而至"，并介绍朱熹格物穷理之说。王阳明深受触动。 | 关键转折：从"圣人或为天生"转向"圣人可学"，为毕生追问圣学路径埋下动因。娄谅亦传递了朱熹"即物穷理"的方法论，为后来"格竹"实验提供直接背景。 |
| 约 1489–1492 | "格竹"实验：依朱熹格物说，对着庭院竹子沉思穷理七日，未得"物理"，反而病倒。此后对朱熹外向格物路径产生深度怀疑。 | 思想史上著名的失败实验。标志着阳明与程朱路线的第一次实践性碰壁，为日后转向"心即理"积累了切身体验。 |
| 约 1492–1499 | 广泛涉猎佛学（禅宗尤甚）与道家（炼养之术），曾一度沉浸其中，后自述"出入佛老"数年。 | 接触内向性修行传统，为"良知"内在论提供了思想养料，也使他日后能够在批判佛老的同时保留其内在工夫的合理成分。 |
| 1492 | 第一次参加乡试，落第。 | — |
| 1495 | 第二次参加会试，落第。据载同考者多有沮丧，阳明反劝慰众人"不以落第为耻，以落第动心为耻"[存疑：此话确切场景或有文学加工]。 | 折射出早期"不动心"工夫的雏形，与后来"致良知"中不随外境动摇的精神一脉相承。 |
| 1499 | 第三次参加会试，中进士（二甲第七名）。时年27岁。进入仕途。 | 科举正途完成，思想探索并未因此停歇——入仕反而提供了将学问付诸实践的场所。 |

## 二、仕途与思想探索（1499–1506）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1499–1500 | 授刑部主事，参与审判。因体弱，奉命出游南京，接触当地文人圈，文学诗赋创作活跃。 | 初入官场，体验律法与人心的张力；文学创作期与思想沉淀并行。 |
| 约 1500–1504 | 转任兵部主事，研习兵法韬略，对边防、屯田等军事政务颇有钻研。此期间持续"出入佛老"，同时未放弃对儒学圣道的寻求。 | 军事实务经验为后来平宁王之乱等军事行动打下基础；内心仍处于儒/佛/道三重张力之中。 |
| 约 1502–1503 | 赴阳明洞（绍兴会稽山）筑室修炼，据传修得"预知"之能（坐忘）[存疑：此类超常叙述多为后人附会]，但随即自我反省，认为此乃"著相"，转回儒学立场。 | "阳明"之号即源于此。此次修炼与放弃，标志着他最终放弃以佛道为出路，重新将儒家圣学定为根本方向。 |
| 1504 | 在北京主持乡试，开始公开讲学，强调圣学，与当时流行的记诵词章之学分庭抗礼。弟子逐渐聚集。 | 首次以教师身份出现，心学传播初现雏形；对"学以成圣"的倡导开始影响士人圈。 |
| 1505–1506 | 孝宗崩，武宗（正德帝）即位，宦官刘瑾擅权，朝政急剧恶化。阳明对朝廷风气深感忧虑。 | 政治环境骤变，即将引发阳明人生最重要的转折点。 |

## 三、龙场悟道（1506–1510）

| 年份 | 事件 | 思想意义 |
|------|------|---------|
| 1506 冬 | 因上疏为戴铣、薄彦徽等言官鸣冤（弹劾刘瑾），被廷杖四十，下锦衣卫狱，后贬为贵州龙场驿丞（正九品）。沿途刘瑾派人追杀，阳明伪造投水自尽痕迹脱身 [存疑：追杀之说见于部分文献，真实性存争议]。 | 生死考验。廷杖与流贬成为"事上磨练"的极端形式，也是将抽象道德信念付诸现实代价的第一次大考。 |
| 1507–1508 | 抵达贵州龙场（今修文县）。龙场地处西南荒僻，多瘴疠，少汉人，衣食匮乏。阳明自建草庐，以夷族语言与当地苗族沟通，教化周边村民。 | 从中原文明秩序跌入"化外之地"，剥除了一切社会身份与文化庇护，构成哲学突破的物质条件。 |
| 1508 | **龙场大悟**（史称"龙场悟道"）：某夜从石棺（自制石椁，日夜静坐其中）中顿悟，高呼"圣人之道，吾性自足，向之求理于事物者误也"。悟出"心即理"——理不在外物，而在吾心之中。 | **阳明心学核心命题的诞生**。与朱熹"即物穷理"彻底分道：理不需向外格物求取，心本身即是理的根源。这是中国哲学史上"内转"最具戏剧性的时刻之一。 |
| 1508–1509 | 在龙场开始讲学，提出**"知行合一"**：知与行非两件事，真知必然导向行动，不行即非真知。为当地官民讲授，并著《五经臆说》[存疑：此书今已不传，内容主要见于转述]。 | "知行合一"直接批判了程朱"先知后行"的两段论，消解了道德知识与道德行为的外在分裂，将工夫论与本体论统一。 |
| 1510 | 刘瑾被诛（武宗正德五年），阳明获平反，升任庐陵（今江西吉安）知县，后转南京刑部主事、吏部主事等职。 | 政治回春；龙场经历使其实践工夫与讲学风格更具感召力，弟子数量迅速扩张。 |

## 四、仕途回升与讲学（1510–1519）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1510–1514 | 历任庐陵知县、南京刑部主事、太仆寺少卿等职。边治政边讲学，弟子遍布江西、浙江、南京等地。主要弟子如徐爱（王阳明最早入室弟子，早逝）、薛侃等开始记录其语录。 | 心学从个人顿悟转型为有组织的讲学运动；《传习录》上卷主要内容形成于此期（经徐爱等整理）。 |
| 约 1514–1516 | 升南京太仆寺少卿、鸿胪寺卿。在南京讲学影响日盛，与湛若水（甘泉）深度往来，二人既相互欣赏又在"心"与"随处体认天理"上有学术分歧 [存疑：二人分歧的深度历来有不同解读]。 | 心学与同时代其他新儒学流派（如甘泉学）的对话与区分，促进了阳明思想的进一步精确化。 |
| 1516 | 升都察院左佥都御史，奉命巡抚南赣（江西南部、福建汀州、广东潮州、湖广郴州等地），主持剿抚盗贼流民。 | 从讲堂进入军政前线，开始将心学工夫大规模运用于治理实践（乡约、保甲、义学等制度创设）。 |
| 1517–1519 | 在南赣推行**"南赣乡约"**，建立地方保甲、社学制度；先后平定闽赣粤边境数股匪患。军事行动与教化并重，被称为"攻心为上"的治理典范。 | "事上磨练"的集中体现。阳明明确反对将军事与道德分离：平乱本质是恢复人心秩序，而非单纯军事征服。 |
| 1519 年 6–7 月 | **宁王之乱**：宁王朱宸濠在南昌起兵，号称十万大军，意图夺位。阳明当时正赴福建途中，得报后就地募兵，以不足万人之众，用疑兵之计、水战奇袭，**43天内**（一说35天 [存疑]）平定叛乱，生擒宁王。 | 军事奇迹背后是心学"致良知"的实践逻辑：临机决断不依赖外部程序，而依赖当下良知的清明判断。此役确立了阳明作为当代第一流军事家兼思想家的历史地位。 |
| 1519 年 秋 | 平乱后遭嫉，宦官张忠、许泰率军南下，欲夺功甚至构陷阳明。阳明被迫将生擒的宁王转交，忍辱自保，期间数度面临政治危机。 | "处变不惊"：良知在极度政治压力下的稳定性再次被验证。此段经历后阳明对"致良知"的表述更加成熟。 |

## 五、晚年与思想成熟（1519–1529）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 约 1520–1521 | 在江西、浙江讲学期间，**"致良知"**概念被明确提出并系统化。阳明后来自述："吾良知二字，自龙场以后，便已不出此意，只是点此二字不出……五十岁方与学者直说此话。" | **心学三大命题（心即理、知行合一、致良知）完整确立**。"致良知"综合了本体（良知即天理）与工夫（致，即扩充、实践），被阳明本人视为"千古圣传之秘"。 |
| 1521 | 武宗驾崩，世宗嘉靖即位，阳明获封新建伯（世袭），加官进爵，但因政治斗争（与杨廷和等人矛盾）而逐渐被排挤出核心权力圈，此后数年主要在绍兴讲学。 | 功名达至顶峰，却选择以讲学为重心——印证"功名非志"的心学立场。 |
| 1521–1527 | 在绍兴（稽山书院等地）大规模讲学，门人弟子逾千，遍及各阶层（士农工商）。《传习录》中卷、下卷陆续形成（经钱德洪等整理）。主要弟子：**王畿（龙溪）、钱德洪、邹守益、聂豹、王艮**等。 | 心学运动的鼎盛期。"满街都是圣人"等命题在此期讲学中提出，标志着圣学门槛的根本性平民化。 |
| 1527 年秋 | **天泉证道**（天泉桥之辩）：阳明赴广西前夕，弟子王畿与钱德洪在天泉桥就"四句教"展开辩论，请阳明裁决。阳明提出著名的**"四句教"**："无善无恶心之体，有善有恶意之动，知善知恶是良知，为善去恶是格物。"并言明此四句有"接引学者"与"本体直悟"两层用法，嘱二人各守其分 [存疑：天泉证道的具体对话及四句教的确切文字，见于王畿《天泉证道记》，是否完全忠实于原貌有学术争议]。 | 心学本体论与工夫论之间张力的集中呈现。四句教此后成为阳明后学各派分歧的核心文本：王畿主"四无"（本体即无善无恶），钱德洪主"四有"（工夫须为善去恶），两种解读引发明代后期心学内部的持续论争。 |
| 1527–1528 | 以两广总督身份赴广西，平定思恩、田州土司叛乱（卢苏、王受之乱），再度以"攻心"策略（招抚为主、军事为辅）成功平乱，并在当地推行教化。 | 暮年带病出征，"知行合一"与"事上磨练"的最后一次大规模实践，也是心学"经世致用"面向的终极证明。 |
| 1529 年 1 月 | 自广西撤军返浙途中，病情急剧恶化（肺病 [一说痨疾]）。行至江西青龙铺（今大余县），病逝，享年 57 岁。临终，弟子周积问有何遗言，阳明答：**"此心光明，亦复何言！"** | "此心光明"是心学最简洁的终极自证：无需外物、后人或历史来证明，良知在当下即是完整的。临终语成为阳明心学精神的永恒象征。 |
| 1529 年（身后） | 初谥"文毅" [存疑：此谥或系嘉靖朝政争所限，未获应得尊崇]，万历年间（约 1584 年）从祀孔庙，追谥**"文成"**。 | 从生前受压制到身后入祀孔庙，折射出心学在明代的政治命运曲线。 |

## 六、身后影响

| 年份 / 时期 | 事件 | 思想意义 |
|------------|------|----------|
| 1529 年后：弟子三派 | 阳明后学迅速分化为三大主要流派：**浙中派**（以王畿/龙溪为代表，主张"四无"，强调本体直悟，近禅）；**江右派**（以聂豹、邹守益为代表，强调主静归寂与工夫积累，偏保守）；**泰州派**（以王艮为代表，面向平民百姓，讲"百姓日用即道"，后衍生李贽等激进思想家）。 | 三派分歧折射出心学内在的多向张力：本体即工夫 vs. 本体工夫二分；精英士大夫心学 vs. 平民心学；道德保守主义 vs. 思想解放主义。阳明心学由此成为明代后期最具生命力也最具争议的思想运动。 |
| 16 世纪中后期 | 心学在明代中后期广泛扩散，影响文学（公安派"性灵说"）、政治（东林党等）、民间宗教等领域。与此同时，正统程朱理学对心学的批判也持续加剧（如嘉靖、隆庆年间多次"禁讲学"事件）。 | 心学的社会化与政治化：从哲学运动演变为广义的文化-社会变革力量，同时也引发了传统秩序的防御性反弹。 |
| 约 1584 | 神宗万历年间从祀孔庙，追谥"文成"，阳明正式获得儒家道统中的崇高地位。 | 官方认可来得迟，象征着心学与国家意识形态之间长达半个世纪的紧张关系的部分和解。 |
| 17 世纪初：日本 | **中江藤树**（1608–1648）将阳明学系统引入日本，被称为"日本阳明学之祖"。其弟子熊泽蕃山进一步发展，阳明学成为江户时代重要思想流派之一，后影响明治维新志士（如吉田松阴、西乡隆盛等人均受阳明学影响 [存疑：影响程度与直接关联在学界有不同评估]）。 | 阳明学的"东传"：在日本与武士道精神结合，形成强调行动意志与道德自决的独特形态，与中国阳明学的走向出现分叉。 |
| 17–18 世纪 | 清代考据学兴起，以"空谈心性、不务实学"为由对阳明学（及整个明代心学）进行系统性批判（代表人物：顾炎武、黄宗羲虽出自阳明门下但亦有反思，戴震等更对"以理杀人"式道德观提出批判）。黄宗羲著《明儒学案》，是研究阳明及其后学最重要的历史文献。 | 心学在清代的政治压制与学术清算，但也通过黄宗羲等人的整理而得到历史性保存与批判性传承。 |
| 19 世纪末–20 世纪初：民国 | 梁启超、章太炎、王国维等人重新诠释阳明学，将其与民族自强、个人意志联系；**孙中山**、**蒋介石**均公开表达对阳明学的推崇（蒋介石将台湾草山改名"阳明山"）[存疑：二人对阳明学的理解深度与动机，学界评价不一]。日本侵华期间，部分中国知识分子以阳明学为精神动员资源。 | 阳明学的"政治化复兴"：良知自决与知行合一被转化为民族主义与意志论话语，与原初哲学内涵既有传承也有变形。 |
| 20 世纪后半叶至今 | **当代新儒家**（牟宗三、唐君毅等）对阳明心学进行系统的现代哲学诠释，将其与康德道德哲学对话，提出"良知坎陷说"等重要命题。大陆学界自 1980 年代后阳明研究复兴，近年"企业家阳明热"（稻盛和夫哲学与阳明学的比较等）形成新的通俗传播热潮。 | 阳明学的持续生命力：从精英哲学到通俗管理学，从东亚儒学对话到世界哲学比较，阳明心学依然是理解中国内向性道德哲学传统的核心文本。 |
