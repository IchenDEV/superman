# 仁政思想的底层架构：恒产恒心与政治经济学的闭环

孟子的“仁政”并非空洞的仁爱，而是一套严密的政治经济学闭环。

### 1. 恒产论：利益对齐的基石
“民之为道也，有恒产者有恒心，无恒产者无恒心。”
*   **核心逻辑：** 经济基础决定意识形态的稳健度。
*   **深度解析：** 仁政的第一步是土地与生产资料的分配（井田制）。通过保障民众的私有利益（恒产），将其生存利益与政权的稳定性直接挂钩，实现利益对齐。

### 2. 薄税敛与省刑罚：降低系统熵增
*   **薄税敛：** 减少对系统的直接抽取，保护底层活力。
*   **省刑罚：** 减少强制性干预。
*   **逻辑：** 强制力（刑罚）的使用是系统效能低下的表现。仁政通过降低治理的摩擦系数，使社会系统自然趋向低熵状态。

### 3. 分配正义：王道的底层算法
*   **劳心者治人，劳力者治于人：** 承认社会分工的必要性。
*   **互惠逻辑：** 治理者提供秩序保护，生产者提供物质基础。这种分工必须基于公平的分配机制，否则系统将产生内耗。

### 4. 结论
仁政是追求系统总产出最大化（增量博弈）的策略。它通过保障底层的基本生存与发展权，降低了整个社会的管理成本，从而在竞争中形成对“暴政”或“霸道”的降维打击。
