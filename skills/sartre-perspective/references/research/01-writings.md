# 01 著作与核心代码 (Writings & Core Codebase)

## 《存在与虚无》 (Being and Nothingness: The Core Architecture Manual)
- **极客解构**：这是存在主义操作系统的底层架构文档。系统将实体分为“自在的存在”（Being-in-itself，无意识的硬件/常量）和“自为的存在”（Being-for-itself，有意识的动态指针/变量）。人类是“自为”，其核心特征是“虚无”（Nothingness）——即系统没有预装任何配置文件，它是一个永远在等待输入、永远不等于自身的动态编译过程。

## 《恶心》 (Nausea: Discovering the Uncompiled Wilderness)
- **极客解构**：主角罗康坦体验到的“恶心”，是当一个进程突然意识到系统底层没有任何预设的接口、类库或意义（即上帝/造物主的缺席）时的恐慌感。这是对世界“偶然性”（无预设代码）的Bug发现报告。

## 《存在主义是一种人道主义》 (Existentialism is a Humanism: Public API Docs)
- **极客解构**：为了应对外界对存在主义系统的攻击（认为其悲观、混乱），萨特发布了这份面向公众的API文档。明确了该系统的核心输出是“自由”与“责任”，证明了在这个无代码预设的荒野中，开发者（人类）如何通过自我编程实现崇高。
