## 💎 Gold Standard：3D 蒸馏金标准

本项目所有 Skill 均已完成从「平面画像」到「3D 立体系统」的进化，严格遵循以下标准：

*   **🎭 3D 立体侧写 (3D Profile)**：
    *   **生理局限 (Physiological Limits)**：注入人物的肉身约束（如曹操的头风、尼采的偏头痛、图灵的长跑冷却），分析生理状态如何驱动决策阈值。
    *   **内在张力 (Internal Tensions)**：揭示人物核心价值观之间的摩擦（如孔子的理想 vs 现实、加缪的抽象正义 vs 具体母爱）。
*   **🔗 全量引用链 (Full Reference Chain)**：
    *   `SKILL.md` 中的每一个心智模型与决策证据均通过 `[Ref: 0x-xxx.md]` 精准指向项目内部的 `references/research/` 原始调研笔记。**拒绝 AI 虚构，确保护航决策的严谨性。**
*   **🧬 动态表达 DNA**：
    *   根据人物的 **生命周期 (Life Stages)** 自动切换语气（如曹操的青年忧愤、壮年雄心与晚年冷峻自辩）。

## 🚀 快速开始 (Quick Start)

### 1. 安装 (Installation)

#### **Skills CLI (推荐)**
支持一键安装库中所有视角到本地：
```bash
npx skills add IchenDEV/superman
```

#### **Claude Code**
在 `claude` 会话中运行：
```bash
/plugin install IchenDEV/superman
```

#### **Gemini CLI**
运行：
```bash
gemini extensions install https://github.com/IchenDEV/superman
```

### 2. 唤醒视角 (Invoke)
直接在对话中指明您需要的视角。**视角激活后，AI 将直接进入该人物的 3D 认知模式。**

> **User:** *"中本聪，如何评价目前基于 PoS 的中心化趋势？"*
>
> **Satoshi:** *"If you don't believe me or don't get it, I don't have time to try to convince you, sorry. 权力不应来自质押，而应来自可验证的数学共识。PoS 终将导致系统回落到旧有的官僚塔中……"*

## 🏮 方法论溯源 (Methodology)

本项目的心智模型蒸馏技术深度遵循 **[Nuwa (女娲)](https://github.com/alchaincyf/nuwa-skill)** 标准：

1. **考古式调研**：拒绝百科，只采集一手文献、访谈与原始决策记录。
2. **3D 模型重构**：将人物的生理局限（如眼疾、失眠、痛感）作为其思维框架的底层动力。
3. **数据驱动透明**：通过 `references/research/` 目录公开所有的蒸馏过程与论据。
