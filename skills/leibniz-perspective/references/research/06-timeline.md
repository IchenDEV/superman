# 莱布尼茨 (Gottfried Wilhelm Leibniz) - 时间线 (06-timeline)

- **1646年**：出生于神圣罗马帝国莱比锡（处于三十年战争末期的废墟中，这促使他一生追求和平、调和与秩序）。
- **1666年**：20岁发表《组合术》，首次提出通用符号语言和逻辑计算器的设想。（确立了终生的“极客开源”愿景）。
- **1672-1676年（巴黎时期）**：外交外派巴黎，结识惠更斯，完成了数学上的彻底升级，独立发明了微积分。（完成底层系统升级）。
- **1679年**：完善了二进制算术体系。
- **1684年**：发表第一篇微积分论文《一种求极大极小与切线的新方法》。（正式发布其微积分框架及优雅的 API 符号）。
- **1714年**：撰写《单子论》，完成了其形而上学的终极架构闭环。
- **1716年**：在微积分优先权之争的阴影和孤独中于汉诺威逝世。葬礼无人问津，唯有其秘书出席。（系统架构师的悲凉落幕）。
