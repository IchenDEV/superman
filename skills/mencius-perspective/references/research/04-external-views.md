# 孟子的雄辩术：排比逻辑、道德势能与类推陷阱

孟子的雄辩术是基于“浩然之气”的心理攻势与逻辑压制，将道德高地转化为言语势能。

### 1. 排比逻辑：波浪式的情感叠加
*   **层递式推进：** 孟子的排比不是词藻堆砌，而是通过节奏感造成压迫。
*   **压缩选择空间：** 如《鱼我所欲也》，通过不断假设与否定，将受众引入唯一的逻辑出口——舍生取义。听者在心理上会不自觉解除防御。

### 2. 势能构建：道德重力场
*   **造势：** 从不纠缠细节，直接拉升维度。
*   **重力引导：** 以性善论为原点，一旦对手承认同理心，就如同进入了孟子的重力场。随后的推论顺着道德坡度倾泻而下，形成“以气御言”的绝对自信。

### 3. 类类推逻辑：引君入瓮
*   **常识套牢：** 用对方无法反驳的常识套住其政治行为。
*   **逻辑推恩：** 如对齐宣王，从“以羊易牛”的不忍之心出发。既然你不忍看牛死，为何能看百姓苦？这种逻辑平移让对手无法否认结论，因为否定结论就等于否定了自己的善意。

### 4. 辩论策略：归谬与反客为主
*   **归谬法：** 将对方观点推向极端以显其荒谬（如反驳墨家兼爱为“无父无君”）。
*   **反客为主：** 用提问代替回答，通过一连串追问牵引对手逻辑，直到对手说出孟子预设的答案。
