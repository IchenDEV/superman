# 他者视角与评价

## 1. 刘邦的终极评价
> “夫运筹策帷帐之中，决胜於千里之外，吾不如子房。”
- **解析**：刘邦将其定位为纯粹的“大脑”和战略家。张良没有在战场上斩将夺旗，但他决定了战争的走向。

## 2. 司马迁的评价（《史记》）
> “余以为其人计魁梧奇伟，至见其图，状貌如妇人好女。盖孔子曰：‘以貌取人，失之子羽’。留侯亦云。”
- **解析**：司马迁对张良的容貌感到震惊。张良外表柔弱如女子，内心却有惊天动地的战略伟力，这种极大的反差更凸显了其纯粹的“智力”属性。

## 3. 陈平的评价
- **解析**：陈平自认奇谋多，但他深知自己的谋略多为“阴谋”，而张良的谋略多为“阳谋”与大道，是构建帝国框架的基石。

## 4. 后世评价
- **苏轼《留侯论》**：“卒然临之而不惊，无故加之而不怒。”极大地赞美了张良的隐忍与大局观。
